- `>=0.85` → `auto_routed` (state=`'routed'`)
- `0.50-0.85` → `review_pending` (state=`'classified'`)
- `<0.50` → `failed`

**WHEN and HOW does the user/system review and CONFIRM the auto_routed classifications?**

- Or does the user only see the `review_pending` items?
- At what point are routed docs "trusted" for downstream calculations? (i.e., when does a routed doc become a confirmed input to the calc engine, per ADR-001 confirmation-sacred?)
